M0228RA1 - Identifica l’entorn de desenvolupament web

Anàlisi d’un projecte web real (WordPress o fet a mida):

  • Identifica les tecnologies utilitzades (HTML, JS, PHP, CSS…)
  • Determina si és client-server, SPA, etc.
  • Explica com es comunica amb el servidor i amb quina arquitectura.

Evidència: Informe tècnic de 2 pàgines amb esquemes i conclusions.

M0228RA2 - Dissenya interfícies web

Activitat: Prototipat i maquetació d’una pàgina principal d’aplicació web.

  • Utilitza HTML i CSS (amb o sense Bootstrap).
  • Ha de ser responsive i amb bones pràctiques d’accessibilitat.

Evidència:

  • Fitxer .html
  • Captures i enllaç de desplegament si es publica
  • Breu vídeo de 2 minuts explicant el disseny

M0228RA3 – Implementa la lògica del client (JS)

Activitat:

Afegeix interactivitat amb JS pur (o amb jQuery si es treballa).
  • Validació de formularis
  • DOM manipulació (mostrar/ocultar elements, tabs, etc.)
  • Esdeveniments d’usuari

Evidència:

  • Codi + demostració (vídeo curt o fitxer ZIP)
  • Breu documentació tècnica del que ha implementat

M0228R4–Connexió amb el servidor

Activitat: Creació d’un formulari amb connexió a servidor (amb PHP) • El formulari ha de guardar dades en un fitxer o una base de dades. • Ha de validar al servidor i retornar un missatge. Evidència: • Codi complet + capturar la interacció client-servidor • Arxiu .sql (si hi ha BBDD)

M0228R5-Control de versions i desplegament bàsic

Activitat: Publicació del projecte web a GitHub i (opcional) Netlify o 000Webhost • Desplegar el frontend (i backend si ho permet). • Incloure README.md amb descripció i instruccions. Evidència: • Enllaç a repositori • Captura del desplegament • Verificació del funcionament bàsic online
